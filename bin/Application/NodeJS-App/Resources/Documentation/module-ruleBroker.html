(inner) processRules(inputData, inputMetaData, rulesToExecute)

Parse the given input Object/String/Integer/Data/Function through a set of business rules, (Some rules do not support chaining); where the rules are defined in the input rules map.

Parameters:
Name Type Description
inputData string | integer | object | function

The primary input data that should be processed by the business rule.
inputMetaData string | integer | boolean | map

Additional meta-data that should be used when processing the business rule.
rulesToExecute map

The name(s) of the rule(s) that should be executed for modding the input data.
