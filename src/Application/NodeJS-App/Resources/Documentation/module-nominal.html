(inner) workflow(inputData, inputMetaData) → {Boolean}

Takes an arguments array where the second array object would contain a workflow name. We will look up the named workflow in the D-data structure and send that workflow to a call to the command sequencer. Which will in-turn convert the list of commands in that workflow into commands that are enqueued to the command queue.
